# План реализации задания 1.10

## Цель

Доработать программу так, чтобы она помогала исследовать системы дифференциальных уравнений из задания:

- находить окрестность нулевого стационарного состояния;
- строить матрицу линеаризации;
- вычислять собственные значения;
- определять тип устойчивости;
- при необходимости показывать фазовый портрет и траектории.

## Системы из задания

### а)

```text
dx/dt = 2xy - x + y
dy/dt = 5x^4 + y^3 + 2x - 3y
```

### б)

```text
dx/dt = x^2 + y^2 - 2x
dy/dt = 3x^2 - x + 3y
```

### в)

```text
dx/dt = e^(x + 2y) - cos(3x)
dy/dt = sqrt(4 + 8x) - 2e^y
```

### г)

```text
dx/dt = ln(4y + e^(-3x))
dy/dt = 2y - 1 + cbrt(1 - 6x)
```

## Математический план

1. Для каждой системы описать правую часть `f(x, y), g(x, y)`.
2. Проверить, что точка `(0, 0)` является стационарной:

```text
f(0, 0) = 0
g(0, 0) = 0
```

3. Найти матрицу Якоби:

```text
J = [ df/dx  df/dy ]
    [ dg/dx  dg/dy ]
```

4. Подставить `(0, 0)` и получить матрицу линеаризации.
5. Найти собственные значения матрицы 2x2 через след и определитель:

```text
lambda^2 - tr(J) lambda + det(J) = 0
```

6. Определить устойчивость:

- `det < 0` — седло, неустойчиво;
- `det > 0`, `tr < 0`, дискриминант `> 0` — устойчивый узел;
- `det > 0`, `tr > 0`, дискриминант `> 0` — неустойчивый узел;
- `det > 0`, `tr < 0`, дискриминант `< 0` — устойчивый фокус;
- `det > 0`, `tr > 0`, дискриминант `< 0` — неустойчивый фокус;
- `det > 0`, `tr = 0`, дискриминант `< 0` — центр в линейном приближении;
- случаи с нулевыми собственными значениями требуют отдельного анализа.
