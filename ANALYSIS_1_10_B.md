# Задание 1.10 б

Исходная система:

```text
dx/dt = x^2 + y^2 - 2x
dy/dt = 3x^2 - x + 3y
```

## 1. Стационарные состояния

Стационарные состояния находятся из системы:

```text
x^2 + y^2 - 2x = 0
3x^2 - x + 3y = 0
```

Из второго уравнения:

```text
y = x/3 - x^2
```

Подставим в первое уравнение:

```text
x^2 + (x/3 - x^2)^2 - 2x = 0
```

После преобразования:

```text
x * (x^3 - 2/3*x^2 + 10/9*x - 2) = 0
```

Отсюда получаем две действительные стационарные точки:

```text
P1 = (0, 0)
P2 ≈ (1.172997, -0.984922)
```

## 2. Линеаризация

Обозначим:

```text
f(x,y) = x^2 + y^2 - 2x
g(x,y) = 3x^2 - x + 3y
```

Матрица Якоби:

```text
J(x,y) = [ df/dx  df/dy ]
         [ dg/dx  dg/dy ]
```

Частные производные:

```text
df/dx = 2x - 2
df/dy = 2y
dg/dx = 6x - 1
dg/dy = 3
```

Значит:

```text
J(x,y) = [ 2x - 2   2y ]
         [ 6x - 1    3 ]
```

## 3. Анализ устойчивости

### Точка P1 = (0, 0)

```text
J(0,0) = [ -2   0 ]
         [ -1   3 ]
```

Линеаризованная система:

```text
dx/dt = -2x
dy/dt = -x + 3y
```

Характеристическое уравнение:

```text
(-2 - lambda)(3 - lambda) = 0
```

Собственные значения:

```text
lambda1 = -2
lambda2 = 3
```

Так как одно собственное значение отрицательное, а другое положительное, точка является седлом.

```text
Тип: седло
Устойчивость: неустойчиво
```

### Точка P2 ≈ (1.172997, -0.984922)

Матрица Якоби:

```text
J(P2) ≈ [ 0.345993   -1.969845 ]
        [ 6.037980    3        ]
```

След, определитель и дискриминант:

```text
tr(J) ≈ 3.345993
det(J) ≈ 12.931864
D ≈ -40.531786
```

Собственные значения комплексные:

```text
lambda1 ≈ 1.672997 - 3.183234i
lambda2 ≈ 1.672997 + 3.183234i
```

Действительная часть положительная, значит траектории закручиваются от стационарной точки.

```text
Тип: неустойчивый фокус
Устойчивость: неустойчиво
```

## 4. Фазовый портрет

Фазовый портрет строится программой численно. При клике в большом окне выбранная точка используется как начальное условие, затем траектория считается методом Рунге-Кутты.

В программе доступны три режима:

```text
1 - фазовый портрет исходной нелинейной системы
2 - фазовый портрет линеаризованной системы около P1
3 - фазовый портрет линеаризованной системы около P2
```

Стационарные точки отмечаются красными крестиками. Подписи около точек содержат координаты и тип устойчивости. Клавиша `H` скрывает или показывает эти подписи.

Окна программы:

```text
Phase portrait (x,y)       - фазовая траектория
x(t) for system 1.10 b     - график x(t)
y(t) for system 1.10 b     - график y(t)
```
