# Задание 1.10 б

Исходная система:

```text
dx/dt = x^2 + y^2 - 2x
dy/dt = 3x^2 - x + 3y
```

Проверим нулевое стационарное состояние:

```text
f(0,0) = 0^2 + 0^2 - 2*0 = 0
g(0,0) = 3*0^2 - 0 + 3*0 = 0
```

Значит, точка `(0, 0)` является стационарной.

Матрица Якоби:

```text
J(x,y) = [ df/dx  df/dy ]
         [ dg/dx  dg/dy ]
```

Частные производные:

```text
df/dx = 2x - 2
df/dy = 2y
dg/dx = 6x - 1
dg/dy = 3
```

В точке `(0, 0)`:

```text
J(0,0) = [ -2   0 ]
         [ -1   3 ]
```

Линеаризованная система:

```text
dx/dt = -2x
dy/dt = -x + 3y
```

Характеристическое уравнение:

```text
det(J - lambda E) = 0

| -2 - lambda      0      |
| -1               3-lambda | = 0

(-2 - lambda)(3 - lambda) = 0
```

Собственные значения:

```text
lambda1 = -2
lambda2 = 3
```

Так как одно собственное значение отрицательное, а другое положительное, стационарная точка является седлом.

Итог:

```text
Тип стационарной точки: седло.
Устойчивость: неустойчиво.
```
